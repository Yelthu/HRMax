## Context.
That's a MERN stack(SaSS) application.
HRMax is a full-featured and fully responsive HR management system designed to streamline various HR processes, including Attendance, Duty Roster Management, Leave Management,
KPI (Key Performance Indicator), and User Management modules. 
The system aims to enhance the efficiency of HR operations and provide a seamless experience for HR professionals and employees.
